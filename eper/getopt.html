parse(OptSpecList, CmdLine) -> term()

Parse the command line options and arguments returning a list of tuples and/or atoms using the Erlang convention for sending options to a function.

usage(OptSpecList, ProgramName) -> term()

Show a message on stdout indicating the command line options and arguments that are supported by the program.

usage(OptSpecList, ProgramName, CmdLineTail) -> term()

Show a message on stdout indicating the command line options and arguments that are supported by the program. The CmdLineTail argument is a string that is added to the end of the usage command line.

usage(OptSpecList, ProgramName, CmdLineTail, OptionsTail) -> term()

Show a message on stdout indicating the command line options and arguments that are supported by the program. The CmdLineTail and OptionsTail arguments are a string that is added to the end of the usage command line and a list of tuples that are added to the end of the options' help lines.
