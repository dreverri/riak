riak_kv_cache_backend

riak_kv_cache_backend is a backend that turns a bucket into a memcached-type memory cache.

riak_kv_cache_backend is a backend that turns a bucket into a memcached-type memory cache. Ejects the least recently used objects either when cache gets full or the object's lease expires.

Config Settings

* 'riak_kv_cache_backend_memory' - Specifies the amount of maximum amount of memory allocated to cache, in MB. * 'riak_kv_cache_backend_ttl' - When an object is accessed, renew the lease for this many seconds. * 'riak_kv_cache_backend_max_ttl' - Don't allow the object's lease to be renewed after this many seconds.
