riak_kv_stat

riak_kv_stat is a long-lived gen_server process for aggregating stats about the Riak node on which it is runing.

riak_kv_stat is a long-lived gen_server process for aggregating stats about the Riak node on which it is runing.

Update each stat with the exported function update/1. Modify the internal function update/3 to add storage for new stats.

Get the latest aggregation of stats with the exported function get_stats/0. Modify the internal function produce_stats/1 to change how stats are represented.

Riak will start riak_kv_stat for you, if you have specified {riak_kv_stat, true} in your config .erlenv file.
